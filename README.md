其中有三個參數：key為加密用的金鑰、data為要被加密的資料內容、nonce為在串流加密上只使用一次的隨機數，用於確保安全。
![image](https://github.com/user-attachments/assets/6020e3f4-75cb-414d-a661-16a7799db0cd)
